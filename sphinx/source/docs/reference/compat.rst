Compatibility Layers
====================

Bokeh currently utilizes `mplexporter <https://github.com/mpld3/mplexporter>`_ to
help convert Matplotlib plots into Bokeh plots. MPL plots using features not
supported by mplexporter may not render fully. However, there are plans for
Matplotlib to adopt a native JSON ingest/export functionality of its own in the
future. (`MEP 25 <https://github.com/matplotlib/matplotlib/wiki/MEP25>`_). This
will allow Bokeh and other systems to interact more robustly with matplotlib.

Current MPL compatibility is sufficient to render `Seaborn <http://web.stanford.edu/~mwaskom/software/seaborn/>`_
and `ggplot.py <https://github.com/yhat/ggplot>`_ code. Additionally you can easily
render plots generated by `pandas <http://pandas.pydata.org>`_ as well. You can see
examples of Bokeh rendering plots from all of these libraries in the ``examples/compat``
directory:

* `examples/compat/ggplot <https://github.com/bokeh/bokeh/tree/master/examples/compat/ggplot>`_
* `examples/compat/mpl <https://github.com/bokeh/bokeh/tree/master/examples/compat/mpl>`_
* `examples/compat/pandas <https://github.com/bokeh/bokeh/tree/master/examples/compat/pandas>`_
* `examples/compat/seaborn <https://github.com/bokeh/bokeh/tree/master/examples/compat/seaborn>`_
